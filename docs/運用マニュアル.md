# 月次清算書 自動生成ツール 運用マニュアル

## このツールは何?

毎月の清算書(プログラミング / 将棋 / 文理ヴィクトリー / 速読)を
自動で生成します。源泉 `.xlsm` と Google スプレッドシートを元に、
Y:\(NAS)に対象月の清算書ファイルを書き出します。

---

## 通常運用(何もしなくていい)

**毎月1日 09:00 に自動実行**されます。実行結果は Telegram に自動通知が届きます。

### Telegram 通知の見方

グループ/個人チャットで以下のようなメッセージが届きます:

```
【月次清算書生成】2026-05
成功 4/4

✅ programming: プログラミング売上管理表_202605月分.xlsx
✅ shogi:       スマイル将棋売上管理表_202605月分.xlsx
✅ bunri:       文理ヴィクトリー売上管理表_202605月分.xlsx
✅ sokudoku:    速読_売上管理表_202605月分.xlsx
```

全部 ✅ なら完了。❌ があれば有馬に連絡。

## ファイルを取得する(2通り)

### 方法A: ブラウザから

1. ブラウザで **<http://192.168.1.16:8081/>** を開く
2. ログイン: ユーザー = `miura` / パスワード = (有馬から共有)
3. 画面下の「最近生成されたファイル」から [ダウンロード] をクリック

## 生成後の手動作業(毎月必須)

自動で入らないセルは手入力が必要です。Excel で開いて、**報告書シート** の以下を埋めてください:

| サービス | セル | 内容 |
|---|---|---|
| プログラミング | `AJ3` | 名大SKY直営分マイクラID数 |
| スマイル将棋 | `L6` | 名大SKY 999999 の仕入れ額 |
| 文理ヴィクトリー | 5行目 (K5-P5 等) | 家族ID 999999(名大SKY直営)の各料金 |
| 100万人の速読 | `L6` 等 | 名大SKY直営分の値 |

> 「色のついたセルは関数があるので触らない！」行の下の手入力セルが対象。

---

## 手動で実行したい時(再生成など)

ブラウザで <http://192.168.1.16:8081/> を開いて:

1. 対象月を選ぶ (例: `2026-05`)
2. 対象サービスをチェック(既定は全部ON)
3. Telegram 通知 = ON
4. **[生成実行]** をクリック
5. 画面下のログが「完了 (returncode=0)」になるまで待つ(約3分)
6. 完了後にファイル一覧から [ダウンロード]

> 既存ファイルは上書きされます。消さないでほしいファイルは事前にバックアップを取ってください。

## こんな時は

| 状況 | 対処 |
|---|---|
| Telegram 通知に ❌ が付いた | 有馬に連絡。自分で直さず、元データ(.xlsm)や Google Sheet 側に問題があることが多い |
| ブラウザで Internal Server Error | 有馬に連絡。サービス再起動が必要な可能性 |
| 報告書の数字が明らかにおかしい | 手入力セル(名大SKY直営分など)の漏れ or 元 .xlsm の問題。元ファイルを確認し、必要なら有馬に相談 |
| パスワードを忘れた / 変更したい | 有馬に連絡 |
